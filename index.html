01 — About

Web制作の現場で、
手を動かし続けています。

野崎大翔（Hiroto Nozaki）と申します。デジタルハリウッドSTUDIOでWeb制作を体系的に学び、2025年3月の卒業後すぐに株式会社アストラザスタジオへ入り、Web制作会社で継続的に実務経験を積んでいます。WordPressサイトの制作・更新・修正・ページ追加・デザイン反映を通じて、「作って終わり」ではなく運用され続ける設計の重要性を、現場で身につけてきました。

アストラザスタジオで働き始めてから、「良いサイト」は更新する人がいること、運用に耐えるコードであることまで含めて成立すると知りました。今は企画書から運用想定まで言語化してから手を動かしています。

「焼肉 黒耀」では画像最適化を後回しにし、Lighthouse Performance が大きく落ちました。WebP化・lazy load・preconnect で 90+ まで改善し、以降は次の作品で前作の弱点を潰すサイクルで取り組んでいます。

Vanilla JS で焼肉 黒耀の予約UIを作る中で、状態管理とDOM操作が複雑化していく手触りに違和感が残り、React と TypeScript で再設計を始めました（Project 01+）。型と宣言的UIが設計時点でミスを拾ってくれる体験は、自分のコードを書く前提を変えました。

  • 01

    運用視点で設計する

    勤務先でのWordPress更新・修正対応を通じて、「半年後の自分や他の人が触っても破綻しない構造」を意識するようになりました。命名・コメント・分割の判断軸が変わりました。

  • 02

    企画から書ける

    ペルソナ・KPI・ユーザージャーニー・競合分析まで言語化してからコードに進む手順を、全4作品で意識しています。「なぜこの設計にしたか」を説明できる状態を作るのが習慣です。

  • 03

    自分の作品を批評する

    前作の弱点（画像最適化／インラインstyle／パッチ残存）を次作で潰すサイクルを4作品で回しました。「動くからOK」で止めず、何が悪かったかを言葉にする習慣を続けています。

Currently Learning Next.js / Tailwind CSS / Vitest（テスト） / CI・デプロイ自動化
